Research Experience

Exploring EEG Features for Emotion Recognition: An In-Depth Evaluation with Machine Learning and Deep Learning Approaches

Undergraduate Thesis | 2024

In this research, I evaluated the performance of various feature extraction techniques and classification models for accurately classifying emotions using EEG data. By experimenting with Statistical features, Power Spectral Density, and Differential Entropy (DE), I found DE features to outperform others, achieving an accuracy of 93% using a hybrid CNN-BiLSTM model on the SEED dataset.

Supervisor: S. M. Mahedy Hasan

GitHub
Towards Superior EEG-Based Emotion Recognition: Integrating CNN Outputs with Machine Learning Classifiers for Enhanced Performance

Published on the 6th International Conference on Electrical Engineering and Information & Communication Technology (ICEEICT 2024)

This study explores EEG-based emotion recognition by combining CNNs with machine learning classifiers. Using The SEED dataset, we trained a 2D CNN on Differential Entropy (DE) features and then applied machine learning models to enhance classification.

Authors: Tushar Das, S. M. Mahedy Hasan, Anwar Hossain Efat, Md. Farukuzzaman Faruk, Azmain Yakin Srizon, Md. Al Mamun, Md. Rakib Hossain

GitHub | DOI
Harnessing CNNs for Precise Damage Detection of Asphalt Pavements in Flood-prone Areas

Under Review | 7th International Conference on Advances in Civil Engineering

This work uses CNN models, YOLOv5x and YOLOv8l, to detect pavement distresses like potholes and cracks in flood-prone areas. With a custom dataset of 1500 images, YOLOv8l outperforms YOLOv5x. An innovative feature is the extraction of damage area masks, enabling better assessment of distress severity for prioritizing repairs.
